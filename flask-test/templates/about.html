About this Research Paper Abstract Classifier

This website is part of a project for the Natural Language Processing class from the Master of Data Science at the University of Luxembourg. This website aims at classifiing research paper abstracts into categories based on a NLP Machine Learning model trained on various academic papers. The user inputs an abstract, and the system predicts the category it belongs to.

How it Works:

  • The user enters an abstract text into the form.
  • The system processes the text using a pre-trained vectorizer and model.
  • The predicted category is displayed on the home page.

This website is built using Flask, while the model is mostly built with Scikit-learn.
